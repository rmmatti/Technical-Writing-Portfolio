# Welcome to MRI Leasing Portal User Guide

Write your **leasing operations** efficiently using the *MRI Leasing Portal*. This guide helps users navigate and perform key actions within the system.

## Features

- Tenant onboarding and management  
- Lease creation and tracking  
- Property and unit management  
- Payment and billing overview  
- Reports and analytics  

> This guide is designed for leasing agents, property managers, and administrators.

---
> Process flow: User Login → Dashboard → Add Tenant → Create Lease → Approval → Active Lease → Payment Tracking → Reporting

## 📌 Note for Portfolio Reviewers

 This user guide is created for **demonstration and portfolio purposes only**.  
> The workflows, features, and data presented here are **representative samples** of a real-world leasing platform.  
> In a production environment, the documentation would include **additional modules, detailed configurations, edge cases, integrations, and extended data sets**.

## 1. Login to the Portal

### Steps

1. Open the MRI Leasing Portal URL  
2. Enter your **Username** and **Password**  
3. Click **Login**

> ⚠️ Note: Ensure credentials are correct. Contact admin if login fails.

---

## 2. Dashboard Overview

The dashboard provides a quick view of:

- Active leases  
- Pending approvals  
- Recent activities  
- Notifications  

### Key Widgets

| Widget Name        | Description                          
| Active Leases     | Displays ongoing lease agreements    |
| Notifications     | Alerts for pending tasks             |
| Reports Summary   | Snapshot of leasing performance      |

---

## 3. Tenant Onboarding

### Steps

1. Navigate to **Tenants → Add Tenant**  
2. Enter tenant details:
   - Name  
   - Contact Information  
   - Identification Details  
3. Click **Save**

### Tenant Data Fields

| Field Name     | Description  |              
| Tenant ID      | Unique identifier          |
| Name           | Tenant full name           |
| Email          | Contact email              |
| Phone          | Contact number             |

> ✅ Tip: Ensure all mandatory fields are completed before saving.

---

## 4. Lease Creation

### Steps

1. Go to **Leases → Create Lease**  
2. Select:
   - Tenant  
   - Property  
   - Unit  
3. Enter lease details:
   - Start Date  
   - End Date  
   - Rent Amount  
4. Click **Submit**

---

## 5. Property & Unit Management

### Actions Available

- Add new property  
- Configure units  
- Assign availability status  

### Example Structure

---

## 6. Payments and Billing

### Steps

1. Navigate to **Payments**  
2. View tenant billing details  
3. Track:
   - Rent payments  
   - Due amounts  
   - Payment history  

> ⚠️ Note: Late payments may trigger alerts.

---

## 7. Reports and Analytics

Generate reports for:

- Lease performance  
- Revenue tracking  
- Occupancy rates  

### Steps

1. Go to **Reports**  
2. Select report type  
3. Apply filters  
4. Click **Generate**

---

## 8. Troubleshooting

| Issue                | Solution                          
| Unable to login     | Reset password / contact admin   |
| Data not saving     | Check mandatory fields           |
| Report not loading  | Refresh or check filters         |

## 9. Best Practices
